# srikalyan-marketplace

A curated collection of Claude Code plugins by srikalyan.

## Installation

```bash
# Add the marketplace to Claude Code
/plugin marketplace add srikalyan/srikalyan-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [claude-lcm](https://github.com/srikalyan/claude-lcm) | Lossless Context Management — DAG-backed memory for long agentic sessions |

## Installing a plugin

```bash
/plugin install claude-lcm@srikalyan-marketplace
```
